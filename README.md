# Kazemine-chan

This is just a discord bot I vibe code for fun, and it's working as a confession bot.

## Commands

- `/setup`: Set up channel to confess. **Example**: `/setup #confession-booth`
- `/confess`: Confess. Can be sent in any channels.

## Note

The above commands will work _normally_ if the bot is kept 24/7, but in case it's not, there's a workaround: Deploy it as a web server, then confess on the website
